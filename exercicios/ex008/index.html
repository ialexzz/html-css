Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag "B" (Não Semântica)

Nesta frase, temos um termo em destaque usando o "STRONG" (Semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag "I" (Não Semântica)

Nesta frase, temos um termo em ênfase usando a tag "E" (Semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag "MARK"

E no outro parágrafo, temos outro texto marcado no final

Estamos criando um texto grande e um texto pequeno neste parágrafo "BIG" (NÃO SE USA MAIS)

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado "DEL"

Texto Insiredo

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois "INS" (Semântica)

Existe também o sublinhado com a tag "U" (Não Semântica)

Texto Sobrescrito

Podemos inserir coisas do tipo x20+3 "SUP"
